Project Description

The project aims to design and implement an online restaurant ordering system, using javascript, layui, jquery as the main development technologies for front-end, Spring-Boot for back-end, and MySQL for database. With the development of the catering industry, online restaurant ordering systems are more and more significant for increasing the efficiency of restaurants and improving the convenience of users. In this project, four main improvements will be developed. The first one is showing pictures for each dishes, administrators can upload pictures when editing the menu and then users can see them on the menu. The second one is that administrators can set coupon, users can receive it and use them. The third one is users can add reviews after finishing the order which will show to the restaurant. The last one is that users can see allergens of each dish in the details page which can remind them to avoid ordering the dish contains their allergens or disliked foods.
